Architectural Concept Index

  • JWT Stateless Auth: Token signed by Auth service is decoded and verified locally by Task service.
  • RBAC Rules: Deletion operations are gated by a role check (interns block, admins pass).
  • REST API Design: Standardized query structures are mapped to pagination indices, filters, and sorts.
  • Winston Logger: High performance logs are written to physical log streams on the backend.
